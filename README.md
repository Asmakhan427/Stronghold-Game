# Stronghold Strategy Game

## Overview

**Stronghold Strategy Game** is a console-based multiplayer medieval kingdom simulation developed in **C++** using Object-Oriented Programming principles. Players manage every aspect of a kingdom, including the economy, military, politics, resources, banking, and population while making strategic decisions that influence the survival of their civilization.

The project emphasizes software design and advanced C++ concepts through a modular architecture and interconnected game mechanics.

---

## Features

- Kingdom management simulation
- Army recruitment and training
- Population and social class management
- Resource and supply chain management
- Dynamic economy and taxation system

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/Stronghold-Strategy-Game.git
```

2. Open **Stronghold2.sln** in Visual Studio.

3. Build the solution.

4. Run the project.
